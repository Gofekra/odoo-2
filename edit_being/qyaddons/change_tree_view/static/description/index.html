A Smart User Interface

You can change the tree view from interface without more coding

Just click to add and remove any trees column.

You can add any field in the selected form to be part from the tree as a new column by clicking “add new item” or delete it.

Control the Sequence Of Tree Columns

You can arrange and control the sequence of the columns which will appear in the tree.

Manage your own tree view

Every user who has the privilege of custom tree can keep his tree view of specific form which will not affect the others user view.

Security

All security points were taken in our consideration like:- 1- Customer tree feature is under check box in user form, so specific users who can do that. 2- Every user can see the fields which is related to his privilege.

Reporting

More than 100000+ reports are provided by Custom tree module, as the user can customize the tree then export it to excel sheet or PDF.
